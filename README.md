
# Prework Study Guide Webpage

## Description

The purpose of this project is to create a study guide to learn how to integrate HTML, CSS, JavaScript, Git and JavaScript into a functioning webpage.

## Usage

To use this Prework Study Guide, you can review the notes in each section. For suggestions on what to study first, open the Chrome DevTools using the Inspect feature. There you will see a list of topics learned in the prework along with a suggestion of what to study first.
